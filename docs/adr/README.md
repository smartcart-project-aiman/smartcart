# Architecture Decision Records

This directory contains the Architecture Decision Records (ADRs) for SmartCart.

ADRs document significant technical decisions: what was decided, why,
what alternatives were considered, and what the consequences are.

## Statuses

| Status | Meaning |
|--------|---------|
| 🔄 Pending | Decision not yet made |
| ✅ Accepted | Decision made and active |
| ⬜ Superseded | Replaced by a newer ADR |
| ❌ Deprecated | No longer relevant |
